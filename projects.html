精彩呈现 Exciting Presentation

  • 项目1：波士顿房价多元线性模型 - 根据房价与各特征的相关性分析，选择确定影响房价的主要特征，随机划分数据集，80%样本作为训练数据，20%样本作为测试数据，用训练数据训练波士顿房价线性回归模型，查看各项指标合理性，测试模型在测试集上的性能，对未知数据进行房价预测。

    Project 1: Boston House Price Multivariate Linear Model - Based on the correlation analysis between housing prices and various features, select the main features that affect housing prices, randomly divide the dataset, use 80% of the samples as training data and 20% of the samples as testing data, use the training data to train the Boston housing price linear regression model, check the rationality of various indicators, test the performance of the model on the testing set, and predict housing prices on unknown data.

  • 项目2：主成分分析 - ‌主成分分析（PCA）是一种统计方法，旨在通过线性变换将高维数据转化为低维数据，同时尽可能保留数据的变异性‌。PCA的核心思想是通过找到数据中的主成分（即新的变量），这些主成分是原始变量的线性组合，且彼此之间互不相关，从而在降维的同时保留最重要的信息。

    Project 2: Principal Component Analysis - Principal Component Analysis (PCA) is a statistical method aimed at transforming high-dimensional data into low dimensional data through linear transformation while preserving the variability of the data as much as possible. The core idea of PCA is to find the principal components (i.e. new variables) in the data, which are linear combinations of the original variables and are uncorrelated with each other, thus preserving the most important information while reducing dimensionality.

  • 项目3：时间序列分析 - 时间序列是指按时间顺序排列的一组数据点，通常用于研究某一现象随时间的变化规律。时间序列分析的基本要素包括时间间隔（如日、月、年等）、数据点（各时间点的观测值）以及序列的长度（时间点的数量）。其研究目的主要是理解数据的动态特性，识别潜在的模式和趋势，并进行预测和决策。

    Project 3: Time Series Analysis - A time series refers to a set of data points arranged in chronological order, usually used to study the changing patterns of a phenomenon over time. The basic elements of time series analysis include time intervals (such as days, months, years, etc.), data points (observations at each time point), and the length of the sequence (the number of time points). The main purpose of its research is to understand the dynamic characteristics of data, identify potential patterns and trends, and make predictions and decisions.
